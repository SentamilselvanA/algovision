# 🚀 Quick Guide: Using Phase 2 Features

## Getting Started

### 1. Start the Application
```bash
# Terminal 1 - Backend
cd backend
npm run dev

# Terminal 2 - Frontend
cd frontend
npm run dev

# Open http://localhost:3000
```

### 2. Navigate to Any Algorithm
- Click "Algorithms" in navbar
- Select any algorithm (e.g., Binary Search)

## New Features Overview

### 🎯 Top Bar Controls

**[Problem] Button**
- Click to show/hide sample problem
- See problem statement, examples, and connection

**[Code] Button**
- Click to show/hide code panel
- Toggle code view on/off

### 📝 Code Panel (Bottom Center)

**Language Selector**
- [JavaScript] [Python] [C++]
- Click any language to switch
- Code updates instantly

**Active Line Highlighting**
- Yellow background = currently executing line
- Follows visualization steps automatically
- Synced with animation

### 💡 "Why This Step?" Section (Right Panel)

**Lightbulb Icon**
- Appears when code line is active
- Explains WHY the code does what it does
- Beginner-friendly language

### 📋 Sample Problem (Left Panel)

**When "Problem" is toggled ON:**
- Problem statement
- Example input/output
- How it connects to visualization

## Step-by-Step Usage

### Example: Learning Binary Search

1. **Open Binary Search**
   - Navigate to Algorithms → Binary Search

2. **View the Problem**
   - Click [Problem] button
   - Read: "Find target in sorted array"
   - See example: arr = [2,5,8,12,16], target = 12
   - Understand the connection

3. **Enable Code View**
   - Click [Code] button
   - Code panel appears at bottom
   - Default language: JavaScript

4. **Start Visualization**
   - Click Play ▶
   - Watch visualization animate
   - See code line highlight (yellow)
   - Read "Why this step?" explanation

5. **Switch Languages**
   - Click [Python] or [C++]
   - Code updates instantly
   - Same logic, different syntax

6. **Step Through Manually**
   - Click Next ►
   - Observe code line change
   - Read new "Why" explanation
   - Understand each step deeply

7. **Try Custom Target**
   - Enter new target value
   - Click refresh button
   - Watch algorithm search for it

## Feature Combinations

### Best Learning Experience:
1. **Problem + Code + Visualization**
   - Toggle both [Problem] and [Code] ON
   - See complete picture
   - Understand problem → code → execution

2. **Multi-Language Comparison**
   - Run visualization once
   - Switch between JS/Python/C++
   - See how same algorithm looks different

3. **Slow Motion + Why**
   - Set speed to 1x (slowest)
   - Read "Why" for each step
   - Build deep understanding

## Troubleshooting

### Code not highlighting?
- Check if [Code] button is enabled
- Ensure visualization is running
- Try resetting (↻ button)

### "Why" not showing?
- Some steps may not have explanations
- Check right panel for lightbulb icon
- Try different algorithm

### Language switch not working?
- Click language button again
- Refresh page if needed
- All algorithms support 3 languages
